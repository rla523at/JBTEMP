## 정의
`집합(set = collection = family)`이란 서로 뚜렷이 구분되는 원소들의 모임이다.


## 첨수 집합

$i \in I$에 대해 집합 $S_i$가 대응될 때 $i$를 `첨수(index)`, $I$를 `첨수 집합(index set)` 이라고 한다.
