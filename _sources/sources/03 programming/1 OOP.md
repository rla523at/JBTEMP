# Object Oriented Programming
객체지향(Object Oriented Programming, OOP)란 코드로 구현할 시스템이 있을 때, 이 시스템이 객체들로 구성되어 있으며 시스템의 기능은 객체들의 상호작용에 의해 이루어진다고 보고 프로그래밍을 하는 패러다임이다.
