# Mark Down

## Collapsible Text

Collapsible Text 를 사용하고 싶으면 다음과 같이 하면 된다.

```
<details>
<summary> 제목 </summary>
내용
</details>
```

<details>
<summary> 제목 </summary>
내용
</details>

</br></br>

만약 제목을 Heading 형태로 만들기 위해서는 추가 태그를 붙이면 된다.

```
<details>
<summary> <h2 style="display:inline-block"> 제목 </h2></summary>
내용
</details>
```

<details>
<summary> <h2 style="display:inline-block"> 제목 </h2></summary>
내용
</details>
